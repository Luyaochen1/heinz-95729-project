In the NancyBootstrapper.cs, there are two scopes: application scope (ConfigureApplicationContainer) and request (ConfigureRequestContainer) scope. The application scope is executed once per app start. The request scope is executed once per request, and gives us a place to inject request specific dependencies, such as a user's locale and authorization/security.

The compositionRoot.js is loaded once per page load. Since our app is a Single Page Application (SPA), that means it may be loaded only once per session. We have to account for the fact that the user may refresh the page at any time, though.
